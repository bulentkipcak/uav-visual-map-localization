# Example Logs

These CSV files are the curated main accuracy logs used for thesis/report
analysis.

```text
sift_observe_100m_hover_60s.csv
sift_observe_100m_move_1ms_90s.csv
```

They contain Gazebo ground truth fields and SIFT localization outputs, so they
can be used to reproduce the main accuracy metrics and figures generated under
`chapter4_outputs/`.

Other local experiment logs are intentionally ignored to keep the public
repository reviewable.
