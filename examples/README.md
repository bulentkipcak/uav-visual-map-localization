# Examples

This directory contains small, curated artifacts that are useful for reviewing
the thesis-version results without committing the full local experiment archive.

Large generated map assets, full tile databases, and most intermediate logs are
kept outside Git.

Contents:

- `logs/`: main thesis accuracy logs with Gazebo ground truth.
- `experimental_logs/`: selected EKF/VPE/Kalman integration logs kept as
  supporting evidence and limitations, not as the main success result.
